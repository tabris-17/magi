Telegram

The app-wide Telegram notification bot. The token and chat ID are stored once in magi's settings database (data/magi.db) and shared by every function — Betelgeuse and any future function send through this single bot.

Setup steps:
1. Open Telegram and search for @BotFather → type /newbot → follow prompts → copy the bot token.
2. Start a chat with your new bot by sending it /start.
3. Paste the token below, click Auto-detect for the Chat ID, then Save.
